### src/thousand/wordscramble
> 단어의  랜덤하게 섞고 나서 맞추는 게임입니다.

< 요구 사항 >
1. 단어를 랜덤하게 선택해야 한다.
2. 단어의 스팰링을 랜덤하게 바꿔야 한다.
3. 문제를 맞춰도 계속 진행해야한다.
4. q를 누르면 프로그램이 종료된다.
5. 문제를 틀리면 한 단어씩 힌트로 보여준다.
